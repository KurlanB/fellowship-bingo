# Fellowship Bingo

A small Bingo web app built to play with fellows from the **McCall MacBain Fellowship** during the **interview phase**—great for filling downtime / waiting time between sessions.

## Live Demo
- https://fellowbingo.vercel.app

## Features
- Play a simple Bingo-style game in the browser
- Fast local dev workflow

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
